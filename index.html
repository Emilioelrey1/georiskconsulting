01 — MAPA INTERACTIVO
◉

02 — ANÁLISIS DE RUTAS
⟶
3 Variantes de Ruta
Compara Ruta Directa, Rodeo Norte y Rodeo Sur con score de riesgo ML (0–100) por cada opción.
03 — DASHBOARD
▦
KPIs y Estadísticas
Distribución de incidentes por zona, hora y tipo. Tendencias temporales y análisis de hotspots.
04 — REPORTES PDF
⬗
Reporte de Riesgo
Genera reportes PDF con snapshot de mapa, análisis de rutas e indicadores de riesgo.
05 — ML ENGINE
⬡
Random Forest Score
Modelo scikit-learn entrenado con datos históricos. Considera hora del día, día de semana y densidad zonal.
06 — POSTPIS / API
◈
Backend Geoespacial
Node.js + Express + PostGIS. ST_DWithin para consultas de buffer. API REST documentada.
